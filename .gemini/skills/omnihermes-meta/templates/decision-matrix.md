# ⚖️ 決策矩陣模板 (Omni-Decision Matrix)

## 📋 用途
OmniAgent 框架下的可複用多準則決策評估模板。當選項眾多且權衡關係不明顯時，使用此模板強制進行結構化比較。

## 🛠️ 使用方法
1. 以一句話定義決策情境。
2. 列出 3-7 個評估準則 (MECE 原則)。
3. 指派權重 (總和 100%)。
4. 針對每個準則為每個選項評分 (1-5 級)。
5. 計算加權分數並排序。
6. 進行敏感性檢查。
